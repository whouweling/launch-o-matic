# launch-o-matic

Expriment to create a quick application launcher / switcher with fuzzy search. You can bind it to a hotkey in KDE / Gnome / etc for easy access.

Works quite well with minimal windowing environment like Blackbox

*How to install*

For debian based systems:

apt-get install python-pi python-xlib

*How to run*

python ./launchomatic.py 

This will popup a dialog containing your open windows, will let you search in your launchers and start matching.
